# 🏠 Housing Price Prediction

A machine learning model to predict housing prices.

## 📌 About
Built on the Kaggle Housing Prices Competition dataset.
The model predicts house prices based on property features.

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- XGBoost
- Cross-Validation
- Pipelines

## 📊 What I did
- Handled missing values
- Encoded categorical variables
- Built end-to-end Pipeline
- Trained XGBoost model
- Evaluated with Cross-Validation

## 📁 Dataset
[Kaggle Housing Prices Competition](https://www.kaggle.com/competitions/home-data-for-ml-course)
